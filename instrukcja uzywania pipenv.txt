instalajca pipenv zeby odpalac jego uzywac
https://medium.com/@mahmudahsan/how-to-use-python-pipenv-in-mac-and-windows-1c6dc87b403e

instalacja paczek po przez
pipenv install nazwapaczki

uruchamianie albo
pipenv run python plik

odpalanie virtualnego srodowiska po przez
pipenv shell
